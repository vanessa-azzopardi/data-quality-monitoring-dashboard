# Data Quality Monitoring Dashboard

## Overview

This project demonstrates the design and development of a Power BI dashboard used to monitor tourism survey data collection progress, data quality, interviewer performance, quota achievement, and sample representativeness throughout the data collection lifecycle.

The dashboard was developed to provide operational and management teams with visibility of fieldwork progress, helping identify data quality issues, monitor survey performance, and support evidence-based decision making.

## Technologies

* Power BI
* Power Query
* DAX
* Data Modelling
* Survey Data Analysis

## Key Features

### Data Collection Monitoring

* Monitoring of completed, attempted, refused, unreachable and ineligible cases
* Progress tracking against predefined quotas
* Identification of exhausted and exceeded strata
* Breakdown of outcomes by demographic and geographic characteristics

### Operational Performance Monitoring

* Interviewer-level performance monitoring
* Analysis of completion, refusal and non-contact rates
* Identification of fieldwork bottlenecks and operational issues

### Data Quality Monitoring

* Monitoring of missing data rates at questionnaire level
* Identification of questions with elevated refusal or non-response rates
* Continuous assessment of data completeness throughout collection

### Representativeness Analysis

* Benchmarking of collected data against external population statistics
* Comparison of demographic characteristics such as nationality, education level and employment status
* Early detection of potential sampling bias and under-represented population groups

## Dashboard Pages

### Overview

Provides a high-level summary of survey progress, quota attainment and response outcomes.

### Progress Monitoring

Tracks interviewer performance, response outcomes and fieldwork progress across demographic and geographic groups.

### Data Quality & Benchmarking

Monitors missing data patterns and evaluates sample representativeness through comparisons with external benchmark data.
